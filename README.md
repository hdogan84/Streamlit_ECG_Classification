# Streamlit Demo

## Installation:

``` python
Pip install streamlit 
#(pipenv install streamlit dans une environnement virtuel)
```

## How to launch the demo:

1. **Go to the app directory:**

``` bash
# Change directory
cd app
```
2. **Run the `app.py` script:**

``` python
# Run streamlit app
streamlit run app.py
```
